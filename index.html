How It Works

1: Coins Mined
2: Kaspa Funding Wallet receives mined coins.
3: Coins Held until a task or event passes vote for funding.
4: Treasures sign & Sends transfer to funded events wallet address.
5: Funded project/task is completed.

Voting

Votes on spending of Kaspa held in Kaspa Funding  Wallet Address are done through the Kaspa discord using the current voting system that has been utilized by community for crowdfunding efforts past|present.
